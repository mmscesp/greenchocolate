# Cannabis Social Club Platform

Next.js + TypeScript platform for cannabis social club discovery, trust workflows, member requests, and club/admin operations.

## Documentation

All maintained project documentation is now centralized in `docs/`.

- Start here: `docs/README.md`
- Product: `docs/product/`
- Architecture: `docs/architecture/`
- Security: `docs/security/`
- Development: `docs/development/`
- Domain knowledge: `docs/domain/`

## Run Locally

```bash
npm install
npm run dev
```

## Core Commands

- `npm run lint`
- `npm run test:run`
- `npm run build`
